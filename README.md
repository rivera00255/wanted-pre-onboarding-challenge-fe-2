# wanted-pre-onboarding-challenge-fe-1

React, Typescript, Recoil, React Query, axios  
styled components, react hook form

#### 1. 실행방법

```
npm run dev
```

#### 2. 기능

- 로그인 후 todo(할 일 목록) 열람, 작성, 수정, 삭제
- /auth 경로에서 로그인 및 회원가입  
   → 등록되지 않은 사용자의 경우 회원가입  
   → 등록된 회원인 경우 로그인
- 로그인되면 todo 목록으로 이동
- 작성된 todo의 제목 및 내용을 담은 목록 조회
- todo 클릭 시 상세 페이지로 이동  
   → 수정 버튼 클릭 후 제목 및 내용 수정  
   → 삭제 버튼을 클릭하고 확인창에서 수락하면 todo 삭제
